It uses a 3-layer memory architecture (raw captures → structured artifacts → approved work items), executes multi-agent async DAG pipelines triggered from use cases, and tracks costs/tokens per stage with ReAct step tracing. Currently in active development with pipeline report generation, cost tracking, and DOM optimization work mostly complete.
