[PROJECT CONTEXT: aicli]
aicli is a unified LLM project memory system that gives Claude, OpenAI, DeepSeek, Gemini, and Grok the same codebase context, work items, and decisions across CLI, web UI, and IDE integrations. It features a 3-layer memory architecture (raw captures → structured facts → work items), async DAG pipeline orchestration, pgvector semantic search, and role-based agent workflows all driven by YAML configuration.
Stack: language_cli=Python 3.12, cli_framework=prompt_toolkit + rich, backend_framework=FastAPI + uvicorn, backend_auth=JWT (python-jose + bcrypt) + DEV_MODE toggle, database=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
