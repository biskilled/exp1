[PROJECT CONTEXT: aicli]
aicli gives every LLM the same project memory across Claude Code, CLI, Cursor, and web UI by maintaining a unified knowledge base backed by PostgreSQL with pgvector embeddings. The project implements a 3-layer memory architecture (raw captures → structured artifacts → work items), multi-LLM workflow execution (Claude/OpenAI/DeepSeek/Gemini/Grok), role-based agents with YAML configuration, and pipeline-driven task automation with semantic search.
Stack: language_cli=Python 3.12, cli_framework=prompt_toolkit + rich, backend_framework=FastAPI + uvicorn, backend_auth=JWT (python-jose + bcrypt) + DEV_MODE toggle, database=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
