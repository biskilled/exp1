[PROJECT CONTEXT: aicli]
aicli is a unified LLM memory system that enables Claude Code, the aicli CLI, Cursor, and a web UI to share project context across multiple AI sessions. It combines a 3-layer memory architecture (raw captures → structured artifacts → approved work items), multi-LLM workflow pipelines, role-based agent execution, and semantic search via pgvector embeddings.
Stack: language_cli=Python 3.12, cli_framework=prompt_toolkit + rich, backend_framework=FastAPI + uvicorn, backend_auth=JWT (python-jose + bcrypt) + DEV_MODE toggle, database=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
