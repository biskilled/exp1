[PROJECT CONTEXT: aicli]
aicli is a multi-agent LLM platform that gives every AI (Claude Code, CLI, Cursor, web UI) shared project memory via a unified work-item backlog, semantic search, and 4-stage async DAG pipelines. The system syncs commits, prompts, and code changes into PostgreSQL with pgvector embeddings, auto-classifies work items, and routes them to role-based YAML pipelines (Product Manager → Architect → Developer → Reviewer). It supports multiple LLM providers (Claude, OpenAI, DeepSeek, Gemini, Grok) with configurable system prompts and MCP server integration for local tool access.
Stack: language_cli=Python 3.12, cli_framework=prompt_toolkit + rich, backend_framework=FastAPI + uvicorn, backend_auth=JWT (python-jose + bcrypt) + DEV_MODE toggle, database=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
