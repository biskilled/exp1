[PROJECT CONTEXT: aicli]
aicli is a unified LLM memory platform that keeps Claude Code, the aicli CLI, Cursor, and a web UI in sync—sharing the same codebase context, decisions, and feature history. It includes a 3-layer memory architecture (raw captures → structured artifacts → approved work items), multi-LLM workflow pipelines with DAG execution, role-based agents defined in YAML, semantic search via pgvector, and MCP server integration for tools.
Stack: language_cli=Python 3.12, cli_framework=prompt_toolkit + rich, backend_framework=FastAPI + uvicorn, backend_auth=JWT (python-jose + bcrypt) + DEV_MODE toggle, database=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
