[PROJECT CONTEXT: aicli]
aicli is a unified AI project memory platform that allows Claude Code, Cursor, CLI, and web UI to share the same codebase context, decisions, and feature history. It features a 3-layer memory architecture (raw captures → structured artifacts → approved work items), role-based multi-agent pipeline execution with temperature tuning per node, and semantic search over pgvector embeddings of approved work items only. Current focus is on pipeline execution UI, role activation management, and provider-agnostic temperature parameter propagation across all LLM adapters.
Stack: language_cli=Python 3.12, cli_framework=prompt_toolkit + rich, backend_framework=FastAPI + uvicorn, backend_auth=JWT (python-jose + bcrypt) + DEV_MODE toggle, database=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
