[PROJECT CONTEXT: aicli]
aicli is a unified LLM memory platform that gives every AI (Claude Code, CLI, Cursor, web UI) the same project context. It features a 3-layer memory architecture (raw captures → structured facts → approved work items), YAML-driven agent roles with database persistence, a 4-agent DAG workflow pipeline, and semantic search over project events via pgvector.
