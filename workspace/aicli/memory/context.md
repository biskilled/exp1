[PROJECT CONTEXT: aicli]
aicli is a unified LLM memory layer that allows Claude Code, the aicli CLI, Cursor, and web UI to share the same project context—eliminating context copy-paste and re-explanation. The memory architecture uses 3 layers (raw captures, structured artifacts, work items) backed by PostgreSQL + pgvector, with a 4-agent pipeline (PM→Architect→Developer→Reviewer) for approved work items.
Stack: cli=Python 3.12 + prompt_toolkit + rich, backend=FastAPI + uvicorn + python-jose + bcrypt + psycopg2, frontend=Vanilla JS + Electron + Vite, ui_components=xterm.js + Monaco editor + Cytoscape.js, storage=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
