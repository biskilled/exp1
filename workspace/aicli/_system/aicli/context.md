[PROJECT CONTEXT: aicli]
aicli is a shared AI memory platform combining a Python FastAPI backend with an Electron desktop UI, designed to capture, analyze, and promote development activities into structured work items. It uses PostgreSQL with pgvector for semantic search, multiple LLM providers (Claude/OpenAI/DeepSeek/Gemini/Grok) for intelligent synthesis, and an async DAG workflow engine to coordinate multi-stage agent pipelines. Current focus is on refining memory synthesis (Haiku dual-layer), simplifying schema design (mem_ai_work_items reorganization), and implementing tag suggestion workflows with user approval.
Stack: cli=Python 3.12 + prompt_toolkit + rich, backend=FastAPI + uvicorn + python-jose + bcrypt + psycopg2, frontend=Vanilla JS + Electron shell + Vite dev server, ui_components=xterm.js + Monaco editor + Cytoscape.js + cytoscape-dagre, storage_primary=PostgreSQL 15+ with pgvector (1536-dim, text-embedding-3-small)
